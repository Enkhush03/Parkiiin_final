## 🎯 Зорилго

Энэхүү лабораторийн ажлын зорилго нь HTML болон CSS ашиглан энгийн вэб хуудас зохиох, бүтэц (structure) болон загварчлал (styling)-ын үндсэн ойлголтуудыг практикт хэрэгжүүлэхэд оршино.

---

## 📚 Ашигласан технологи

* HTML5
* CSS3

---

## 🧩 Хийсэн ажлууд

Энэхүү лабораторийн хүрээнд дараах зүйлсийг хэрэгжүүлсэн:

* Вэб хуудасны үндсэн бүтэц (header, main, footer) үүсгэсэн
* Текст, зураг, холбоос (link) оруулсан
* CSS ашиглан:

  * Өнгө (color, background)
  * Фонт (font-family, size)
  * Margin, padding
  * Layout 
* Responsive дизайн

## 💡 Сурсан зүйлс

Энэхүү лабораториор дамжуулан:

* HTML-ийн үндсэн tag-уудыг ашиглаж сурсан
* CSS ашиглан веб хуудсыг загварчлах чадвар эзэмшсэн
* Layout зохион байгуулалт хийх ойлголттой болсон
* Багаар хийхэд тулгарсан алдааг сайжруулсан
